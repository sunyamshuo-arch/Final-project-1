# Final Project - Java GUI Application

## Author
Yanshuo Sun

## Description
This project is a simple Java GUI application built using Swing.  
It creates a window with a label, a text field, and a button.  
The purpose of this project is to practice GUI design and basic Java programming.

## Project Structure
- App.java → Main application file (GUI setup)
- UML.png → UML diagram of the project (to be added)

## How to Run
1. Open the project in VS Code
2. Compile the program:
   javac App.java
3. Run the program:
   java App
